# A-BiLSTM Student Performance Prediction

This repository contains source code for a machine learning research project focused on predicting student academic performance trajectories in KNUST online learning systems.

## Proposed Model

A-BiLSTM (Attention-Augmented Bidirectional Long Short-Term Memory)

## Research Goal

To improve sequential student performance prediction using an engineered deep learning model combining BiLSTM with an attention mechanism.

## Tools and Libraries

* Python
* TensorFlow
* Keras
* NumPy
* Pandas
* Scikit-learn
* Matplotlib
